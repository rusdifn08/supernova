## ✨ Fitur Unggulan

### 🎛️ Tiga Simulasi Produk Interaktif (Live Demo di Halaman)

Pengunjung tidak hanya membaca — mereka **mengoperasikan** produknya langsung:

| Pilar | Produk | Simulasi Interaktif |
|-------|--------|---------------------|
| **1** | **Booking System** | Alur 4 langkah: Pilih Studio → Live Calendar + jam + paket harga → Pembayaran (QRIS/Kartu/E-Wallet) → Sinkronisasi real-time ke sisi **Customer & Admin** di 3 platform sekaligus |
| **2** | **Cashier & Inventory** | Scan QR dengan efek laser → item meluncur ke keranjang POS → stok di database berkurang otomatis (100 → 99) dengan flash hijau |
| **3** | **QR Table Ordering** | Scan QR code realistis (13×13 finder pattern) → pilih menu interaktif (+/− qty) → order masuk **bersamaan** ke Kitchen Display & Cashier Display → dapur: Terima → Masak → Selesai → kasir otomatis menampilkan **"Pesanan Ready!"** → status live di HP customer |

### 🌠 Pengalaman Visual Futuristik

- **Aurora Background Hidup** — 4 blob gradasi (cyan/ungu/magenta) yang *berkelana seperti awan*: translasi ratusan pixel, berganti warna senada, morphing bentuk organik, dan parallax scroll
- **Glassmorphism Premium** — kartu `bg-white/60` + `backdrop-blur-2xl` + shadow berlapis
- **Ikon 3 Platform Melayang** — Desktop, Web, Mobile hadir di **setiap section** dengan posisi & jalur jelajah berbeda
- **Shimmer Text** — highlight judul dengan kilau navy→cyan yang mengalir (loop)
- **3D Tilt Cards** — kartu simulasi merespons posisi mouse dengan rotasi perspektif
- **Scroll Progress Bar** + **Scrollspy Navbar** — indikator underline cyan meluncur antar menu (Difference · Product · ROI · FAQ · Contact) mengikuti posisi scroll secara real-time
- **CTA "Bernafas"** — tombol utama dengan glow berdenyut cyan↔ungu + kilau otomatis melintas berkala

### 💰 Kalkulator ROI (The Financial Closer)

Slider interaktif Rp500.000 – Rp10.000.000/bulan dengan angka animasi count-up yang menunjukkan total uang yang terbuang ke langganan SaaS dalam 5 tahun.

## ⚙️ Kustomisasi Cepat

Semua titik konfigurasi penting berada di bagian atas `app/page.tsx`:

```ts
// Link WhatsApp — ganti sesuai nomor bisnis Anda
const WA_CONSULT_URL = "https://wa.link/y8k6c3";
const WA_ORDER_URL   = "https://wa.me/6285174232225?text=...";
const WA_LEAD_NUMBER = "6285174289548";
```

| Ingin mengubah | Lokasi |
|----------------|--------|
| Warna brand | `tailwind.config.js` → `theme.extend.colors` |
| Menu simulasi QR | `page.tsx` → konstanta `QR_MENU` |
| Studio & paket booking | `page.tsx` → array `studios` & `packages` di `BookingSimulation` |
| Daftar FAQ | `page.tsx` → array `faqs` di `FaqSection` |
| Port server | `package.json` → scripts `dev` / `start` (`-p 3333`) |
